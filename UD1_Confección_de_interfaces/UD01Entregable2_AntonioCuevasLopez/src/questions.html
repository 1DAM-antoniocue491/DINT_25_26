¿Qué es React?

React es una biblioteca de JavaScript desarrollada por Meta (Facebook) para construir interfaces de usuario basadas en componentes reutilizables.

¿Qué es un componente en React?

Un componente es una función o clase que devuelve una parte de la interfaz (UI). Cada componente puede tener su propio estado, propiedades y lógica.

¿Qué son las props en React?

Las props (propiedades) son datos que se envían desde un componente padre a un componente hijo para personalizar su comportamiento o contenido.

¿Qué es el estado (state) en React?

El estado es un conjunto de datos que un componente puede controlar internamente. Cuando el estado cambia, el componente se vuelve a renderizar automáticamente.

¿Qué es el Virtual DOM?

El Virtual DOM es una representación ligera del DOM real en memoria. React lo utiliza para comparar los cambios y actualizar solo las partes necesarias de la interfaz.

¿Qué son los hooks en React?

Los hooks son funciones especiales que permiten usar el estado y otras características de React sin escribir clases. Ejemplo: useState y useEffect.
